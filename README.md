# pld-git-branch
# Team Portfolio Git Branching Workshop

A collaborative portfolio website project created to practice real-world Git workflows, branching strategies, pull requests, merge conflict resolution, and frontend-backend integration.

---

# Project Overview

This workshop simulates a real development team environment where multiple students work on different parts of the same project using Git and GitHub collaboration tools.

The project includes:

- Responsive frontend portfolio website
- Team collaboration using Git branches
- Pull request workflow
- Merge conflict handling
- Simple Python Flask API backend
- Dynamic frontend data loading using Fetch API
